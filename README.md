## About The Project

[![Select Component][product-screenshot]](https://todo-app-one-tawny.vercel.app/)

A Simple Todo Application.

Some of the features are:

- Adding a todo
- Completing a todo
- Removing a todo
- User authentication is required for adding unlimited todos

## Getting Started

To get a local copy of the project and to start running the application, you can follow the instructions below.

### Prerequisites

Before you begin, you will need to have the following tools installed on your machine:

- [Git] (https://git-scm.com)
- [Node.js] (https://nodejs.org/en/).
- In addition, it is good to have an editor to work with the code like [VSCode] (https://code.visualstudio.com/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Rai321han/todo-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
